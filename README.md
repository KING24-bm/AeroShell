💫AeroShell💫

AeroShell is a custom-built terminal with a clean UI and useful tools, designed to feel like a real shell but with your own commands.

Features-
Minimal terminal-style UI with tabs
Custom commands
CPU & RAM monitoring
System & device info
Internet search from terminal (net_search)
Command autocomplete

How to Run-
Download the .exe file(also remember to have the other dll files and the folders with the .exe file or else the EXE file will not run!)
Double-click to launch AeroShell
